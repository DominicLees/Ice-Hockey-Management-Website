# Ice Hockey Team Management Website

This is the repository for my undergrad final project, a website for managing ice hockey teams.

## Stack

- ExpressJS backend
- PugJS server side rendering
- MondoDB database
- Mongoose database schemas
- Gulp.js build script

## Features

- Player and coach accounts
- Users able to be members of multiple teams
- Game sign ups
- Game schedule
- Line builder for coaches, coaches can build 5v5, PP and PK lines
- Line summary for players (summaries who they are changing with, what special team lines they are on)
- Record game results and player stats
- Player profiles display their all time stats
